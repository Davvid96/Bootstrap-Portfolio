Skills

The skills I want to learn from this Bootcamp!

HTML/CSS Javascript API's React

In this module, I want to learn how to build webpages that use HTML and CSS to define their structure and style. I will also be introduced to Git, a version control system that stores your code and its history, as well as separate versions of the code. These building blocks of web development are essential skills and tools that I will use throughout My career.

Foundational JavaScript programming principles allow us to design engaging and interactive user experiences. This part of the Bootcamp will cover the basics of language syntax, conditional statements, arrays, and looping in order to build the fundamentals of programming.

Establishing a strong foundation in JavaScript early will pay dividends in my career as a front-end developer. Libraries like React use JavaScript to build the dynamic, performant web apps that are in-demand across many industries. Future modules will build on the skills I will learn in this module, so it will be important to pay close attention to the lessons and practice exercises I will come across.

In this part of the Bootcamp I am going to lear about API's.
API stands for Application Programming Interface.
A Web API is an application programming interface for the Web.
A Browser API can extend the functionality of a web browser.
A Server API can extend the functionality of a web server.

In this module, I will learn about a new way of managing React state: Hooks. React Hooks allow you to use React by "hooking" into its features, such as state and lifecycle methods, without creating class components. I will also learn how to test React components using the Jest and Enzyme libraries, while learning a new set of practices known as behavior-driven development, or BDD.

Finally, I will deploy my own React application to the web using Netlify and learn to set up a (CI/CD), or continuous integration and continuous deployment, pipeline—a series of steps to ensure that new versions of my application are deployed correctly each time new code is pushed to production.
